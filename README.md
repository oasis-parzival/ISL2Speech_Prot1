# ISL2Speech_Prot1

1️⃣ Setup

Clone the repo & open in VS Code.

Open terminal and run: pip install -r requirements.txt



2️⃣ Run the App

Type: python inference.py

(Press 'q' to quit).



3️⃣ Adding New Signs (Optional) If you want to add a new word:

Create a folder in ISL_Dataset (e.g., "Food") & drop your video in it.

Run python create_dataset.py

Run python train_classifier.py

Run inference.py again to test!
